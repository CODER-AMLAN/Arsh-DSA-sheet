## 283. Move Zeroes

***Problem link***[283. Move Zeroes](https://leetcode.com/problems/move-zeroes/description/)

```java
class Solution {
    public void moveZeroes(int[] nums) {
        int k=0;
        for(int x:nums){
            if(x!=0){
                nums[k] = x;
                k++;
            }
        }
        for(int i=k;i<nums.length;i++){
            nums[i]=0;
        }
    }
}
