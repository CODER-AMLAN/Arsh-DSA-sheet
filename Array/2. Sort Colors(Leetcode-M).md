## 75. Sort Colors

***Problem Link**[75. Sort Colors](https://leetcode.com/problems/sort-colors/description/)

```java
class Solution {
    public void sortColors(int[] nums) {
        int[]arr = new int[3];
        for(int x:nums){
            arr[x]++;
        }
        int k=0;
        for(int i=0;i<3;i++){
            for(int j=0;j<arr[i]&&arr[i]!=0;j++){
                nums[k] = i;
                k++;
            }
        }
    }
}
